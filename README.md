# EvidenceShot

EvidenceShot は、現在のタブを証跡向けに撮影して保存する Chrome 拡張機能です。

## バージョン

1.0.2

## できること

- 表示中の範囲を撮影
- スクロール連結でページ全体を撮影
- 中央本文のみを優先した撮影
- PNG / JPEG / WEBP 形式で保存
- 日付スタンプのデザインと文字サイズを選択
- 左下固定テキストを日付スタンプと同じデザイン / 文字サイズで表示
- ブラウザテーマに合わせてポップアップをライト / ダーク自動切り替え

## 使い方

1. Chrome の拡張機能アイコンから EvidenceShot を開く
2. 保存形式や撮影オプションを選ぶ
3. `このタブを撮影する` を押す
4. 画像が Chrome の既定ダウンロードフォルダに保存される

## 撮影範囲

- `表示中のみ（1画面）`
  - いま見えている範囲だけを保存します。
- `ページ全体（レイアウト込み）`
  - 左右カラムを含む画面全体をスクロール連結します。
- `中央本文のみ（主カラム）`
  - 中央の投稿・記事エリアを優先してスクロール連結します。

## 注意

- 撮影開始はポップアップからのみです。
- `http` / `https` ページのみ撮影できます。
- 常時の `<all_urls>` 権限やフローティングボタンは使用しません。
- 画像は offscreen document で逐次合成します。
