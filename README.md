# EvidenceShot

`EvidenceShot` は、Web テスト向けに現在のタブを撮影して保存する Chrome 拡張機能です。

## できること

- 現在表示中の範囲を撮影
- スクロール連結撮影
- 保存形式: PNG / JPEG / WEBP
- タイムスタンプ表示（デザイン・サイズ選択）
- 左下固定テキストの表示（例: 社外秘 / COPYRIGHT）

## 使い方

1. Chrome の拡張機能アイコンから `EvidenceShot` を開く
2. 必要な設定を選ぶ
3. `このタブを撮影する` を押す
4. 画像は Chrome の既定ダウンロードフォルダに保存される

## 撮影モード

- `表示中のみ（1画面）`
  - いま見えている範囲だけを保存
- `ページ全体（レイアウト込み）`
  - 画面全体（左右カラム含む）をスクロール連結
- `中央本文のみ（主カラム）`
  - 中央の主要コンテンツ領域を優先して連結

## スクロール連結の仕様

- 撮影開始時点のスクロール範囲の末尾までを対象にする
- 撮影中に後から増えたコンテンツは対象外

## 注意

- 撮影開始はポップアップからのみ
- `http` / `https` ページのみ撮影可能
- 常時の `<all_urls>` 権限やフローティングボタンは使用しない
